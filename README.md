SMS-Tweak Android Application
=============================

  * About 

SMS-TWEAK is an Android App to process incoming SMS. 

- Multiple filters
- Custom actions 


  * Build 

  git clone git@github.com:ofauchon/smstweak.git
  cd smstweak

  /path/to/sdk/tools/android  update project  --target 1 -p . -n smstweak
  ant debug
