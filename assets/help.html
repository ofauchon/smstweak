Overview

Sms-Tweak filters incoming SMS. When new SMS arrives and matches your rule, an action is performed.

What are the rules?

Rule can be based on caller-id or message body. The rules are written with a simple grammar.

eg: 'callerid equals +33610490000 and body contains HELLO'


Rules parser understand simple operators like 'and', 'or', 'contains', 'equals'... and more

What are the actions?

When a rule matches an SMS, one or more actions can be performed.
Action can be playing custom sound, drop message ... and more

  • popup : Bring the incoming message to the user with a pop-up message
  • play : plays a custom sound
  • vibrate : Vibrate if message matches
  • pass: Forward the incoming message to original sms manager.
